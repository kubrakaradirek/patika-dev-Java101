# ☕ Java101: From Fundamentals to Backend Mastery

This repository is a comprehensive collection of projects, exercises, and homework completed during the **Patika.dev Java101** course. It serves as the foundation of my technical journey in the **n11 TalentHub Backend Bootcamp**.

---

## 📚 Curriculum & Project Roadmap

I have structured my learning path into organized modules, ranging from basic variables to advanced object-oriented programming.

### 1. Basic Concepts & Variables
*Understanding the building blocks of Java.*
- **Data Types:** Byte, Short, Int, Long, Float, Double, Char, Boolean, String.
- **Projects:** - 🧮 Grade Point Average Calculator
  - 🛒 VAT (KDV) Amount Calculator
  - 📐 Hypotenuse & Area of a Triangle
  - 🚕 Taximeter Program
  - 🍎 Green Grocer Cashier Program

### 2. Conditional Statements & Code Blocks
*Logic-driven development and decision making.*
- **Logic:** If/Else, Switch-Case, Comparison & Logical Operators.
- **Projects:**
  - 📱 Simple Calculator
  - 👤 User Authentication System
  - 🌡️ Event Suggestion Based on Weather
  - ✈️ Flight Ticket Price Calculator
  - 🐉 Chinese Zodiac & Leap Year Finders

### 3. Loops (Control Structures)
*Mastering repetitive tasks and mathematical sequences.*
- **Structures:** While, Do-While, For, Break & Continue.
- **Projects:**
  - 🔢 Sum of Odd Numbers & Power of 2 Finder
  - 💎 Armstrong & Harmonic Number Calculators
  - 🏦 ATM Project (Basic Banking)
  - 📉 GCD & LCM (EBOB-EKOK) Finder
  - 🏁 Fibonacci Series & Prime Number Finder

### 4. Methods & Recursion
*Writing modular, reusable, and efficient code.*
- **Concepts:** Return/Void, Overloading, Local Variables, Recursive Methods.
- **Projects:**
  - 🔁 Palindrome Number Checker
  - 🎨 Pattern-Based Recursive Method
  - ⚡ Advanced Power & Prime Functionality

### 5. Object-Oriented Programming (OOP)
*Modeling real-world problems with classes and objects.*
- **Concepts:** Classes, Objects, Constructors, Modeling.
- **Projects:**
  - 🎓 Student Information System
  - 🥊 Boxing Match Simulation
  - 💸 Employee Salary Calculator (Advanced OOP Application)

### 6. Arrays & Data Handling
*Efficient data management using single and multi-dimensional structures.*
- **Concepts:** 1D & 2D Arrays, For-Each, Arrays Class Methods.
- **Projects:**
  - 📊 Max/Min Value Finder
  - 🔄 Array Frequency Analysis
  - 📐 Matrix Transpose Calculator
  - 🔠 Letter Drawing with Multi-dimensional Arrays

### 7. Utility Classes & Final Project
*Finalizing the journey with built-in libraries and logic.*
- **Tools:** Math Class, String Methods.
- **Capstone Project:**
  - 💣 **MineSweeper:** A text-based logic game that integrates all the concepts above.
